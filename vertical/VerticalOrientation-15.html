This file provides a more readable version of VerticalOrientation-15.txt and corresponds to Revision 15 of Unicode® Technical Report #50: Unicode Vertical Text Layout. The character repertoire of this revision is the repertoire of Unicode Version 8.0.

This file is not formally part of the Unicode Character Database at this time.

Each line consists of the following four fields:

  1. Vertical_Orientation (vo) property value, one of the following:
    • U – Upright, the same orientation as in the character code charts
    • R – Rotated 90 degrees clockwise compared to the code charts
    • Tu – Transformed typographically, with fallback to Upright
    • Tr – Transformed typographically, with fallback to Rotated
  2. Unicode code point value or range of code point values in hexadecimal form
  3. Character glyph or glyphs
  4. Character name or names, character block name, or character names list subhead

The characters whose code points are not listed have Vertical_Orientation=R by default.
